## 2024-05-23 - [Interactive vs Static CTAs]
Insight: The application relies heavily on static "Call to Action" banners (e.g., "Request a Check-up"). While visually appealing, they lack immediate value for the user.
Strategy: Replacing static banners with interactive tools (like the Quick Estimator) provides instant utility ("What's the price?") and increases the likelihood of conversion, as verified by the existence of specific regression tests for these features.
